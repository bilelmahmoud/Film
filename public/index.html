API Films

Documentation

Cette API permet d'afficher, d'ajouter, de modifier et de supprimer des films de la base de données. Les routes sont accessibles à partir de l'url suivante :

Routes pour les films

Récupérer la liste de tous les films

GET /api/films

https://film-22gk.onrender.com/api/films

Paramètres possibles :

https://film-22gk.onrender.com/api/films?tri=annee&order=asc

tri
  • titre
  • annee
  • realisation
order
  • asc
  • desc

Récupérer un film spécifique

GET /api/films{id}

https://film-22gk.onrender.com/api/films/{id}

Exemple : id : kcxYu9VvWM0fo2IuGUuq.

Ajouter un nouveau film

POST /api/films

https://film-22gk.onrender.com/api/films

Paramètres requis : un objet film dans le corps de la requête.

const film = {
titre: 'Titre du film',
description: 'Description du film',
annee: 2020,
realisation: 'Nom du réalisateur',
genre: ['Genre 1', 'Genre 2'],
titreVignette: 'Titre de la vignette',
};

Modifier un film existant

PUT /api/films/{id}

https://film-22gk.onrender.com/api/films/{id}

Supprimer un film

DELETE /api/films{id}

https://film-22gk.onrender.com/api/films/{id}

Paramètre requis : id (l'identifiant du film à supprimer).

Routes pour les utilisateurs

Connexion d'un utilisateur

POST /api/utilisateurs/inscription

https://film-22gk.onrender.com/utlisateurs/inscription

Paramètres requis : un objet utilisateur dans le corps de la requête.

const utilisateur = {
courriel: 'test@test.com',
mdp: '12345',
};

Connexion d'un utilisateur

POST /api/utilisateurs/connexion

https://film-22gk.onrender.com/utlisateurs/connexion

Paramètres requis : un objet utilisateur dans le corps de la requête.

const utilisateur = {
courriel: 'test@test.com',
mdp: '12345',
};
